Reminder Setup

  1. Open any goal and enable Set a reminder.
  2. Choose interval: Hourly / Daily / Weekly / Custom / Once.
  3. For Weekly: set day and time.
  4. Allow notifications in iOS Settings when prompted.

Widget Setup

  1. Add Goalio widget from Home Screen edit mode.
  2. Long-press widget and choose your mode/goal.
  3. Tap widget to open the linked goal detail view.
  4. Use quick actions (Check In, +$10, +10%) where shown.

FAQ

Why didn’t I get a reminder?

Check iOS notification permission, Focus modes, and that reminders are enabled globally and for that goal.

Widgets are stale or not updating

Open Goalio once to refresh data, then try re-adding the widget if needed.

How do I sync across devices?

Enable iCloud Sync in Goalio settings and use the same Apple ID on all devices.

Privacy Summary

Goalio stores your goal data on-device. If you enable iCloud Sync, data syncs through Apple CloudKit. Goalio uses notification permission only for reminders you enable and does not sell personal data.
